# Convert Markdown to PDF

This command converts markdown files to PDF format using md-to-pdf. It supports:

- GitHub-flavored markdown
- Syntax highlighting for code blocks
- Custom CSS styling
- Multiple file conversion

## Usage

When invoked, provide the path to the markdown file you want to convert:
- Single file: `/tools:pdf document.md`
- Multiple files: `/tools:pdf file1.md file2.md`
- Pattern: `/tools:pdf docs/*.md`

The PDF will be created in the same directory as the source markdown file, with a `.pdf` extension.
